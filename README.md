# 🛡️ CyberShield - Cybersecurity Awareness Platform

**CyberShield** is a modern, premium interactive web application designed to educate users about common cybersecurity threats and provide real-world tools to improve their digital safety.

## 🔗 [Live Demo - Visit CyberShield](https://lustrous-kangaroo-016631.netlify.app/](https://cybersecurity-awarness-website-using.onrender.com))

---

## 🌟 Key Features

### 🔐 Interactive Security Tools
*   **Password Strength Checker**: Get real-time feedback and crack-time estimations for your passwords.
*   **Phishing Simulator**: Test your skills in identifying malicious emails vs. legitimate ones.
*   **Data Breach Checker**: Simulate a database scan to see if your sensitive information has been leaked.
*   **Advanced Password Generator**: Generate cryptographically secure passwords with custom lengths and character sets.
*   **Security Checklist**: A gamified progress tracker to help you secure all your online accounts.

### 🧠 Educational Components
*   **Cybersecurity Quiz**: Challenge yourself with 10 real-world scenarios and get a detailed performance breakdown.
*   **Threat Gallery**: Deep dives into Phishing, Malware, Social Engineering, and Public Wi-Fi risks with actionable tips.

### 🎨 Design & UX
*   **Premium Dark Theme**: Sleek, modern interface using Glassmorphism and vibrant gradients.
*   **Fully Responsive**: optimized for Desktop, Tablet, and Mobile devices.
*   **Micro-animations**: Smooth transitions and hover effects using vanilla CSS and JavaScript.
